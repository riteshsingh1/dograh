# CodeCanyon Listing Pack

This file contains ready-to-use listing content and asset checklist.

## 1) Product Description (Long)

Collarx is a production-ready voice AI platform for building, deploying, and scaling conversational voice agents with telephony and WebRTC support. It includes a visual workflow builder, campaign dialing orchestration, run analytics, usage dashboards, and multi-provider telephony support.

The package includes:

- Next.js frontend dashboard
- FastAPI backend APIs
- Docker-based deployment stack
- Workflow and campaign management
- Multi-provider telephony integration
- Buyer documentation and deployment guides

Core runtime capabilities are delivered through the `collarx-engine` package and validated using domain-bound license keys.

## 2) Feature Bullets

- Visual voice-agent workflow builder
- Inbound and outbound voice call flows
- Campaign management and dialing orchestration
- Real-time run tracking and reporting
- Multi-provider telephony support
- Pluggable architecture for future integrations/tools
- Domain-bound license validation
- Docker-first deployment
